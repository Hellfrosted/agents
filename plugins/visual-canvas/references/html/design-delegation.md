# Design Delegation

Visual Canvas is a meta-skill. It should not duplicate every taste rule from
specialist design skills.

Use Impeccable when it is available and the artifact has meaningful visual
surface area. Relevant Impeccable concerns include typography, color, layout,
motion, hierarchy, accessibility, anti-slop checks, and visual QA.

If a profile names another design/taste skill, prefer that skill for the
specified concern. Visual Canvas remains responsible for artifact structure,
profile resolution, asset orchestration, and final delivery.

Specialist skills produce guidance, checks, assets, or focused passes. Visual
Canvas integrates those results into the compact artifact pipeline.
