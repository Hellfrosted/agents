Terminal Addons Cheatsheet

One job per addon. When something breaks, find owner first.

Ownership Map

AddonOwnsDoes Not OwnYour Current Hook
Atuin Command history database, history search UI, optional encrypted sync. Does not make Bash history or PSReadLine read from Atuin. Old history files can still update. Ctrl-R; up-arrow disabled; AI binding disabled in PowerShell.
fzf Generic fuzzy picker. Reads list in, lets you filter/select, prints selection out. Does not know shell history unless some wrapper feeds history into it. Bash key-binding files; PowerShell via PSFzf.
PSFzf PowerShell bridge from PSReadLine key chords to fzf. Not installed on WSL Bash; not prompt; not Atuin history. Ctrl-T file/provider picker; Alt-C directory picker.
zoxide Smart directory jump database based on places you visit. File picking, history search, prompt drawing. z query jumps; zi query opens fzf picker.
Starship Cross-shell prompt: OS, shell, git, directory, time, command duration. Line editing, history, file picking. PowerShell 7 and WSL Bash use ~/.config/starship.toml.
Oh My Posh Legacy Windows PowerShell prompt theme. PowerShell 7 prompt in current profile. Windows PowerShell 5.1 profile only.
PSReadLine PowerShell command-line editor, predictions, Tab behavior, key handlers. Not WSL Bash; not fzf itself. Windows edit mode, no bell, history ListView, Tab MenuComplete.
ble.sh WSL Bash line editor: syntax highlighting, enhanced completion, suggestions, keymap behavior. Not command-history database, not prompt renderer, not directory jumper. Sourced after fzf readline hooks and before Starship/zoxide/Atuin in /home/crunch/.bashrc.

Debug First Question

History problem? Check Atuin first, then PSReadLine/Bash history.
File picker problem? Check fzf/PSFzf and fd commands.
Bash editing problem? Check ble.sh before plain Readline; it is the Bash-side PSReadLine equivalent.
z problem? Check zoxide init and database.
Prompt problem? Check Starship or Oh My Posh, depending shell.
